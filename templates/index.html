Project Description

Goal: To create a full stack application that would intake Fitbit or fitness data and pass it through a machine learning model to make predictions.

First, we searched for usefull data. Fitbit user data is not open to the public therefore we decided to use crowdsourced data found here.

Once we took a look at the data we decided to split up responsibilities. Two of our members would work on developing our machine learning models while the rest would focus on building an application.

The first thing we created was a Flask application that would serve our project. Then we quickly developed an html front-end with bootstrap components that would host our application and project description. We wanted the home page to maininly focus on the project description and team roles and credits. On the other hand our form tab would display our projects demo and capabilities. On the front end we focused heavily on the user experience hence the multiple features involved through out the entire application.

In the end we added the machine learning models to our flask application.

For a deeper dive into our application please be sure to check out our project repo, our keynote presentation and our machine learning model notebooks.
